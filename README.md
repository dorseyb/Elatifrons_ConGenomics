This directory contains files to optimize parameters of a Stacks assembly (m,M,n), filter assembled loci, and perform some of the analyses in Dorsey et al. Conservation Genomics Provides Essential Insights into the Provenance of Ex Situ Collections and their Utility for Species Recovery: A Case Study in Encephalartos latifrons for the purpose of each script.
The Elat_analysis and Elat_filtering scripts are not to be called from the terminal. They need to have options and directories set at several points in the code. So, probably best to run in Rstudio.
Please see the methods in Dorsey et al. Conservation Genomics Provides Essential Insights into the Provenance of Ex Situ Collections and their Utility
for Species Recovery: A Case Study in Encephalartos latifrons for the purpose of each script.
